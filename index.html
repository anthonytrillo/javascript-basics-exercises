Ejercicio 1

Verificar si un número es positivo, negativo o cero.

Pasos:

  1. Pedir al usuario que ingrese un número mediante prompt.
  2. Almacenar ese número en una variable descriptiva.
  3. Utilizar una estructura if-else para verificar el valor del número.
  4. Mostrar el resultado en un console.log() o alert().
        
          let numero = Number(prompt("Ingresa un número:"));

          if (numero > 0) {
            console.log("El número es positivo.");
          } else if (numero < 0) {
            console.log("El número es negativo.");
          } else {
            console.log("El número es cero.");
          }

Ejercicio 2

Calcular el descuento aplicable a una compra según el monto total.

Pasos:

  1. Pedir al usuario que ingrese el monto total mediante prompt.
  2. Almacenar ese monto total en una variable descriptiva.
  3. Utilizar una estructura if-else para determinar el descuento según el monto total.
  4. Aplicar el 20% de descuento si el monto es igual o mayor a 1000
  5. Aplicar el 10% de descuento si el monto es igual o mayor a 500 pero menor a 1000
  6. Mostrar el resultado en un console.log() o alert().
        
          let montoTotal = Number(prompt("Ingrese el monto total de la compra:"));

          // Una buena práctica es inicializar la variable descuento con un valor
          let descuento = 0;

          if (montoTotal >= 1000) {
            descuento = 0.2 * montoTotal; // 20% de descuento si el monto es igual o mayor a 1000
          } else if (montoTotal >= 500) {
            descuento = 0.1 * montoTotal; // 10% de descuento si el monto es igual o mayor a 500 pero menor a 1000
          }

          console.log("El descuento aplicable es de: " + descuento + " unidades monetarias.");

Ejercicio 3

Verificar si un número es par o impar.

Pasos:

  1. Solicitar al usuario que ingrese un número entero mediante prompt.
  2. Almacenar el número ingresado en una variable llamada `numero`.
  3. Utilizar el operador de módulo `%` para calcular el resto de la división de `numero` entre 2.
  4. Verificar si el resto es igual a 0.
  5. Si el resto es igual a 0, imprimir en la consola "El número es par".
  6. De lo contrario, imprimir en la consola "El número es impar".
        
          let numero = parseInt(prompt("Ingresa un número:"));

          // Utilizamos el operador de módulo (%) para verificar si el número es par o impar
          if (numero % 2 === 0) {
            console.log("El número es par.");
          } else {
            console.log("El número es impar.");
          }

Ejercicio 4

Determinar si una contraseña ingresada es correcta.

Pasos:

  1. Establecer una contraseña correcta y almacenarla en una variable llamada `contraseniaCorrecta`.
  2. Solicitar al usuario que ingrese una contraseña.
  3. Almacenar la contraseña ingresada en una variable llamada `contraseniaIngresada`.
  4. Comparar si `contraseniaIngresada` es igual a `contraseniaCorrecta`.
  5. Si las contraseñas son iguales, imprimir en la consola "¡Contraseña correcta! Acceso concedido".
  6. De lo contrario, imprimir en la consola "Contraseña incorrecta. Acceso denegado".
        
          const contraseniaCorrecta = "secreto123"; // Contraseña correcta

          let contraseniaIngresada = prompt("Ingresa la contraseña:");

          // Utilizamos una estructura if-else para verificar si la contraseña ingresada es correcta
          if (contraseniaIngresada === contraseniaCorrecta) {
            console.log("¡Contraseña correcta! Acceso concedido.");
          } else {
            console.log("Contraseña incorrecta. Acceso denegado.");
          }

Ejercicio 5

Asignar una calificación según la puntuación obtenida en un examen.

Pasos:

  1. Solicitar al usuario que ingrese la puntuación obtenida en el examen.
  2. Almacenar la puntuación ingresada en una variable llamada puntuacion.
  3. Utilizar una estructura if-else para evaluar la puntuación según los siguientes rangos:
    • Si puntuacion es mayor o igual a 90, asignar la calificación "A".
    • Si puntuacion es mayor o igual a 80, asignar la calificación "B".
    • Si puntuacion es mayor o igual a 70, asignar la calificación "C".
    • Si puntuacion es mayor o igual a 60, asignar la calificación "D".
    • De lo contrario, asignar la calificación "F".
  4. Imprimir en la consola la calificación obtenida.
        
          let puntuacion = Number(prompt("Ingresa la puntuación obtenida:"));

          let calificacion;

          // Utilizamos una estructura if-else para asignar la calificación según la puntuación
          if (puntuacion >= 90) {
            calificacion = "A";
          } else if (puntuacion >= 80) {
            calificacion = "B";
          } else if (puntuacion >= 70) {
            calificacion = "C";
          } else if (puntuacion >= 60) {
            calificacion = "D";
          } else {
            calificacion = "F";
          }

          console.log("La calificación obtenida es: " + calificacion);
