Ejercicio 1

Verificar si un número es positivo, negativo o cero.

Pasos:

  1. Pedir al usuario que ingrese un número mediante prompt.
  2. Almacenar ese número en una variable descriptiva.
  3. Utilizar una estructura if-else para verificar el valor del número.
  4. Mostrar el resultado en un console.log() o alert().
        
          let numero = Number(prompt("Ingresa un número:"));

          if (numero > 0) {
            console.log("El número es positivo.");
          } else if (numero < 0) {
            console.log("El número es negativo.");
          } else {
            console.log("El número es cero.");
          }

Ejercicio 2

Calcular el descuento aplicable a una compra según el monto total.

Pasos:

  1. Pedir al usuario que ingrese el monto total mediante prompt.
  2. Almacenar ese monto total en una variable descriptiva.
  3. Utilizar una estructura if-else para determinar el descuento según el monto total.
  4. Aplicar el 20% de descuento si el monto es igual o mayor a 1000
  5. Aplicar el 10% de descuento si el monto es igual o mayor a 500 pero menor a 1000
  6. Mostrar el resultado en un console.log() o alert().
        
          let montoTotal = Number(prompt("Ingrese el monto total de la compra:"));
          // Una buena práctica es inicializar la variable descuento con un valor
          let descuento = 0;
          if (montoTotal >= 1000) {
            descuento = 0.2 * montoTotal; // 20% de descuento si el monto es igual o mayor a 1000
          } else if (montoTotal >= 500) {
            descuento = 0.1 * montoTotal; // 10% de descuento si el monto es igual o mayor a 500 pero menor a 1000
          }
          console.log("El descuento aplicable es de: " + descuento + " unidades monetarias.");

Juego conteo ascendente

Conteo Ascendente de 2 en 2

Instrucciones:

  1. El juego consiste en un conteo ascendente de 2 en 2.
  2. El jugador debe adivinar el número siguiente en cada iteración del ciclo for.
  3. Si el jugador adivina correctamente, se le dará un punto. Si no, se le restará un punto.
  4. En cada iteración, suma el número actual al valor de suma.
  5. Al final del juego, se mostrará la puntuación obtenida.
        
        
          alert("¡Bienvenido a 'Conteo Ascendente de 2 en 2'!");
          alert("El juego consiste en adivinar los números en el conteo ascendente.");

          let puntuacion = 0;

          for (let i = 0; i <= 10; i += 2) {
            const respuesta = parseInt(prompt("Siguiente número: " + i + ". Adivina el número siguiente:"));

            if (respuesta === i + 2) {
              alert("¡Correcto! Ganaste 1 punto.");
              puntuacion++;
            } else {
              alert("Incorrecto. El número siguiente era: " + (i + 2));
              puntuacion--;
            }
          }

          alert("Fin del juego. Tu puntuación final es: " + puntuacion);

Ejercicio 8

Verificar si un número es Primo.

Pasos:

  1. Define una variable llamada numero y asígnale un número entero positivo mayor que 1.
  2. Declara una variable booleana llamada esPrimo y asígnale el valor true.
  3. Utiliza un ciclo for para iterar desde 2 hasta la mitad del valor de numero.
  4. En cada iteración, verifica si numero es divisible por el número actual del ciclo.
  5. Si numero es divisible por algún número, asigna el valor false a esPrimo y detén el ciclo utilizando break.
  6. Después del ciclo, verifica el valor de esPrimo:
    • Si esPrimo es true, muestra un mensaje indicando que el número es primo.
    • Si esPrimo es false, muestra un mensaje indicando que el número no es primo.
        
        
          const numero = 17;
          let esPrimo = true;

          for (let i = 2; i <= numero / 2; i++) {
            if (numero % i === 0) {
              esPrimo = false;
              break;
            }
          }

          if (esPrimo) {
            console.log(numero + " es un número primo.");
          } else {
            console.log(numero + " no es un número primo.");
          }

          const n = 10;

          let b = 1;

          }

Ejercicio 10

Contador regresivo.

Pasos:

  1. Solicita al usuario que ingrese un número entero positivo.
  2. Inicializa una variable contador con el valor ingresado por el usuario.
  3. Mientras el valor de contador sea mayor o igual a 0:
    • Imprime el valor actual de contador.
    • Decrementa el valor de contador en 1.
  4. Imprime un mensaje indicando que el contador ha finalizado.
        
        
          let numero = parseInt(prompt("Ingresa un número entero positivo:"));
          let contador = numero;

          while (contador >= 0) {
            console.log(contador);
            contador--;
          }

          console.log("¡Contador finalizado!");

Ejercicio 11

Carrito de compras.

Pasos:

  1. Inicializar las variables producto, precioTotal, cantidadTotal y seguirComprando.
  2. Iniciar un ciclo do-while.
  3. Solicitar al usuario que ingrese el producto que desea comprar.
  4. Solicitar al usuario que ingrese la cantidad deseada del producto.
  5. Validar si la cantidad ingresada es un número válido y mayor que cero. En caso contrario, mostrar un mensaje de error y volver al paso 4.
  6. Establecer la variable precio en 0.
  7. Utilizar un switch statement para asignar el precio correspondiente según el producto seleccionado.
  8. Calcular el precio total del producto multiplicando el precio por la cantidad.
  9. Actualizar el precio total acumulativo sumándole el precio total del producto y la cantidad total acumulativa sumándole la cantidad.
  10. Mostrar un mensaje al usuario indicando la cantidad de productos agregados y el total a pagar por el producto actual.
  11. Preguntar al usuario si desea agregar otro producto. Si no desea agregar otro producto, continuar al paso 12. Si desea agregar otro producto, repetir el ciclo desde el paso 3.
  12. Preguntar al usuario si desea finalizar la compra.
  13. Si el usuario decide finalizar la compra, mostrar un mensaje con el resumen de la compra (cantidad total de productos y precio total). Si no desea finalizar la compra, mostrar un mensaje de despedida.
        
        
          let producto = "";
          let precioTotal = 0;
          let cantidadTotal = 0;
          let seguirComprando = true;

          do {
            producto = prompt("¿Querés comprar shampoo, acondicionador o desodorante?", "Ej: shampoo");

            let cantidad = parseInt(prompt("¿Cuántos querés comprar?"));

            while (isNaN(cantidad) || cantidad <= 0) {
              alert("Debés agregar una cantidad válida.");
              cantidad = parseInt(prompt("¿Cuántos querés comprar?"));
            }

            let precio = 0;

            switch (producto) {
              case "shampoo":
                precio = 900;
                break;
              case "acondicionador":
                precio = 1000;
                break;
              case "desodorante":
                precio = 600;
                break;
              default:
                alert("El producto ingresado no es válido");
                continue;
            }

            let precioTotalProducto = precio * cantidad;
            precioTotal += precioTotalProducto;
            cantidadTotal += cantidad;

            alert('Has agregado '+cantidad+' '+producto+'(s)'+' al carrito. Total a pagar: $'+precioTotalProducto);

            seguirComprando = confirm("¿Querés agregar otro producto?");

            if (!seguirComprando) {
              let finalizarCompra = confirm("¿Desea finalizar la compra?");

              if (finalizarCompra) {
                alert('Has comprado un total de '+cantidadTotal+' producto(s)'+' por un total de $'+precioTotal+'. '+' ¡Gracias por tu compra!');
              } else {
                alert("Esperamos volver a verte pronto. ¡Hasta luego!");
              }
            }
          } while (seguirComprando);
